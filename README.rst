authz plugin for `Tutor <https://docs.tutor.edly.io>`__
#######################################################

Authorization Framework for Open edX - Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.con/edunext/tutor-contrib-authz

Usage
*****

.. code-block:: bash

    tutor plugins enable authz
